# stepik-Automation-testing-using-Selenium-and-Python2
Необходимо было:
1. Добавить в файл conftest.py обработчик, который считывает из командной строки параметр language.
2. Реализовать в файле conftest.py логику запуска браузера с указанным языком пользователя. Браузер должен объявляться в фикстуре browser и передаваться в тест как параметр.
3. В файл test_items.py написать тест, который проверяет, что страница товара на сайте содержит кнопку добавления в корзину. Например, можно проверять товар, доступный по http://selenium1py.pythonanywhere.com/catalogue/coders-at-work_207/.
4. Тест должен запускаться с параметром language следующей командой:

pytest --language=es test_items.py       

и проходить успешно. Достаточно, чтобы код работал только для браузера Сhrome.
